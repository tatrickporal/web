Recent Projects

ClosetVirtuoso

ClosetVirtuoso is a website that I and my group with two other members worked on at UCSC's Hackathon held in late January. The challenge was to develop some way of trasferring clothes in a closet into an online system. Thus our project was born. The main focus of the website was to be able to have your own virtual online closet that is unique to the user, and be able to orginize that closet based on color, type of clothing, brands, and be able to save outfits for a later date.

Rasberry PI Python LED

This project was done out of interest and impulse rather than need. In a nutshell the Rasberry PI zero runs a python program I wrote that uses command line arguments to turn on certain LED's connected by a breadboard. If you want more info and specifics click here or just watch the video to see how it works.

The point of this project was to learn certain aspects of what made it work all together. Through this I learned basic soldering, tangable outputs of python, and creating a circuit using a breadboard. All of which will be useful in future projects.

Redtooth

The main goal of this project was to teach myself how CoreBluetooth is implemented in iOS applications. This can be helpful in future projects when using Bluetooth Low Energy (BLE) devices like Bluetooth Arduinos or Rasberry Pi. At this time Redtooth only connects to BLE devices and displays the RSSI number from the peripheral and the central device. Building off the code shown here , I can connect to certian devices (my own laptop, Arduinos, etc) and interact with the services offered with the device.
